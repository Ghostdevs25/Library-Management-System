#Project Title – Library Management System

#Problem Statement

Most libraries do not have automation and use the manual system for such functions as recording of books, tracking of issued copies, it’s members and so on. As its number of books and users continues to increase, such manual approaches become inefficient, time-consuming and error-prone. This project to helps in the efficiency of the library and maintain accurate records.

#Scope of the Project

This is a python based library which can be used to perform following tasks : It contains major core features for add book, register member, issue book and return the same returning book with few records file savedThe controller is currently not supporting GUI level or multi-user role levels but only friendly to a single admin.

#Target Users

Librarians or library administrators

Schools/Colleges of education with small/medium size libraries

People in charge of personal collection of books

Projects of student and laboratory exercises

High-Level Features

Add and search books

Register and manage members

Issue and return books

View items you have checked out, and due dates

Show the values of data previously saved in a structured format
